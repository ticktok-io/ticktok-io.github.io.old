Scheduling

These are the expressions ticktok.io currently supports

Expression Description
every.X.seconds Tick every X seconds
every.X.minutes Tick every X minutes
every.X.hours Tick every X hours
@never This clock will never invoke automatically and it's meant for manual invocation
